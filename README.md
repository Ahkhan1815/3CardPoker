# 3 Card Poker

Overview: A Simple 2-Player 3 card poker game created using JavaFX. 2 Players compete against the dealer and can compete with each other via total winnings.

Purpose: This was a school project and introduced me to Java. It developed my skills in object-oriented and event-driven programming, game logic implementation, and GUI development.

Features:
- Local 2-Player Gameplay
- Polished GUI

What I Learned:
- Java - Programming language used to implement game logic
- JavaFX - Used to develop GUI and handle user interaction
- Maven - Dependency management
